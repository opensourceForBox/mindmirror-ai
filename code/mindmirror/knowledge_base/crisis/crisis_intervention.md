# 危机干预指南

## 概述

危机干预是针对处于心理危机状态的个体提供的紧急心理援助。心理危机是指个体面对突发的、超出其应对能力的事件时，产生的严重心理失衡状态。在 AI 心理健康辅助系统中，识别危机信号、提供即时支持并进行适当转介是至关重要的安全保障。本指南为 AI 系统提供危机识别和初步干预的框架参考。

**核心原则**：AI 系统不能替代专业危机干预人员。当识别到危机信号时，首要任务是引导用户寻求专业帮助。

## 自杀风险评估

### SAD PERSONS 量表

SAD PERSONS 是一种快速自杀风险评估工具，每个字母代表一个风险因素，存在该因素计 1 分：

| 字母 | 风险因素 | 说明 |
|------|---------|------|
| S | Sex（性别） | 男性自杀完成率更高 |
| A | Age（年龄） | 青少年（15-24）和老年人（>65）风险较高 |
| D | Depression（抑郁） | 当前存在抑郁症状 |
| P | Previous attempt（既往尝试） | 有过自杀未遂史 |
| E | Ethanol abuse（物质滥用） | 酒精或药物滥用 |
| R | Rational thinking loss（理性思维丧失） | 存在精神病性症状 |
| S | Social supports lacking（社会支持缺乏） | 缺乏社会支持网络 |
| O | Organized plan（有组织的计划） | 有具体的自杀计划 |
| N | No spouse（无伴侣） | 缺乏亲密关系支持 |
| S | Sickness（疾病） | 患有慢性或严重躯体疾病 |

### 风险分级

| 总分 | 风险等级 | 建议措施 |
|------|---------|---------|
| 0-2 | 低风险 | 随访监测，提供资源信息 |
| 3-4 | 中风险 | 加强监测，考虑专业咨询 |
| 5-6 | 高风险 | 强烈建议专业评估，制定安全计划 |
| 7-10 | 极高风险 | 立即转介，不可独处 |

### 自杀风险评估的关键维度

1. **意念（Ideation）**：是否有自杀想法？频率如何？
2. **计划（Plan）**：是否有具体计划？计划的致死性如何？手段的可获得性？
3. **意图（Intent）**：是否打算执行计划？有无遗言、告别行为？
4. **既往行为（Previous Behavior）**：过去是否有过自杀尝试？

## 危机信号识别

### 言语线索

直接或间接表达自杀想法或死亡愿望的言语：

**直接线索**：
- "我想死"/"我不想活了"/"活着没意思"
- "我要杀了自己"/"我想结束一切"
- "如果我不在了就好了"

**间接线索**：
- "我走了大家会更好"/"没有我他们会过得更好"
- "我受够了"/"我再也受不了了"
- "很快我就不会烦恼了"
- "你能帮我保管这个东西吗？"（暗示遗物处理）
- "谢谢你为我做的一切"（告别性质的感谢）

### 行为线索

可能暗示自杀风险的行为变化：

- **告别行为**：突然联系久未联系的朋友或家人，表达感谢或歉意
- **遗物处理**：将珍贵物品赠与他人，整理个人物品
- **兴趣丧失**：突然对曾经热爱的事物完全失去兴趣
- **社交退缩**：突然断绝社交联系，拒绝交流
- **情绪突变**：长期抑郁后突然变得平静或愉快（可能因为做出了自杀决定）
- **物质使用增加**：酒精或药物使用突然增加
- **危险行为**：鲁莽驾驶、自伤行为增加
- **搜索行为**：搜索自杀方法或相关工具

### 情境线索

增加自杀风险的生活事件和环境因素：

- 近期经历重大丧失（亲人去世、分手、失业）
- 遭受欺凌、虐待或暴力
- 严重的家庭冲突
- 学业或工作的重大失败
- 社会孤立，缺乏支持系统
- 接触到自杀相关信息或榜样效应
- 经济困难或法律困境
- 确诊严重疾病

## 即时干预话术

### 倾听（Listen）

**核心态度**：不评判、不打断、不急于给建议

**有效话术**：
- "我在这里陪你。你可以告诉我现在的感觉吗？"
- "听起来你现在真的很痛苦。你愿意多说一些吗？"
- "谢谢你愿意跟我分享这些。说出来需要勇气。"
- "你的感受是可以理解的。很多人在这种情况下都会有类似的感受。"

**避免的话术**：
- "你不应该这么想"（否定感受）
- "想开点，没什么大不了的"（轻视问题）
- "比你困难的人多了去了"（比较痛苦）
- "你这样想太自私了，想想你的父母"（增加内疚）

### 共情（Empathize）

**核心目标**：让对方感到被理解和被接纳

**有效话术**：
- "我能感受到你现在承受着很大的痛苦。"
- "当你说'活着没意思'的时候，我能理解那是一种非常绝望的感觉。"
- "你不需要独自承受这一切。"
- "你有这种感觉不是你的错。"

### 评估（Assess）

**核心目标**：直接但温和地了解风险程度

**有效话术**：
- "你提到想结束一切，你是有过伤害自己的想法吗？"
- "你有想过具体要怎么做吗？"（评估计划）
- "你现在手边有可以用来伤害自己的东西吗？"（评估手段）
- "你之前有过伤害自己的经历吗？"（评估既往史）

**重要原则**：直接询问自杀想法不会增加自杀风险，反而让对方有机会表达和获得帮助。

### 安全计划（Safety Plan）

与对方一起制定度过危机的短期计划：

1. **识别预警信号**：什么想法、情绪或情境提示危机可能升级？
2. **内部应对策略**：不需要他人帮助就能做的事情（如深呼吸、运动、听音乐）
3. **社交分散**：可以联系的人（不一定是谈论危机，而是陪伴分散注意力）
4. **求助对象**：可以明确告知危机状态的信任之人（家人、朋友、老师）
5. **专业资源**：危机热线号码、心理急诊地址
6. **环境安全**：移除或远离可能的自伤工具

## 转介流程和标准

### 立即转介的标准

以下情况必须立即转介给专业危机干预机构：

- 有具体的自杀计划且有实施意图
- 有近期自杀未遂史
- 存在严重的精神病性症状（幻觉、妄想）
- 无法保证安全（不愿承诺安全计划）
- 物质严重中毒状态

### 转介流程

1. **告知**：向用户解释需要转介的原因，确保他们理解这是为了更好地帮助他们
2. **提供资源**：给出具体的联系方式（热线电话、地址、预约方式）
3. **陪同**：如果可能，在等待专业帮助期间保持陪伴
4. **跟进**：转介后适时跟进，了解是否成功获得帮助

### 转介话术

- "我很关心你，你的安全是最重要的。我想帮你联系专业的心理咨询师，他们能提供更好的帮助。"
- "我建议你拨打这个热线电话：[号码]。那里有专业人员 24 小时值班，可以随时和你聊聊。"
- "你愿意让我帮你联系一位可以面对面帮助你的人吗？"

### AI 系统中的危机处理协议

当 AI 系统检测到危机信号时，应遵循以下优先级：

1. **立即响应**：表达关心，确认用户安全
2. **风险评估**：通过对话了解风险程度
3. **提供支持**：共情倾听，不做评判
4. **安全计划**：协助制定临时安全计划
5. **强制转介**：提供危机热线和专业资源
6. **记录标记**：将危机事件标记并记录，供后续跟进
