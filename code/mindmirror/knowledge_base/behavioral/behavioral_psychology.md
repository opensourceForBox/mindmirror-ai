# 行为心理学

## 概述

行为心理学（Behavioral Psychology）关注可观察的行为及其与环境的关系，认为行为是通过与环境的交互作用学习获得的。行为主义理论为理解人类行为的形成、维持和改变提供了科学框架，也是 CBT 中行为干预技术的重要理论基础。本章涵盖经典条件反射、操作性条件反射、社会学习理论等核心内容，并探讨其在情绪管理和习惯养成中的应用。

## 经典条件反射

### 巴甫洛夫的实验与原理

经典条件反射（Classical Conditioning）由俄国生理学家 Ivan Pavlov 在 20 世纪初发现。在他的著名实验中，狗在听到铃声（原本的中性刺激）后也会分泌唾液，因为铃声与食物（无条件刺激）被反复配对。

### 核心概念

| 术语 | 定义 | 实验中的示例 |
|------|------|-------------|
| 无条件刺激（UCS） | 自然引发反应的刺激 | 食物 |
| 无条件反应（UCR） | 对 UCS 的自然反应 | 分泌唾液 |
| 条件刺激（CS） | 通过与 UCS 配对获得引发反应能力的中性刺激 | 铃声 |
| 条件反应（CR） | 对 CS 的习得反应 | 听到铃声分泌唾液 |

### 关键过程

1. **获得（Acquisition）**：CS 和 UCS 反复配对，条件反应逐渐形成。配对的时间间隔和顺序至关重要——CS 应在 UCS 之前约 0.5 秒出现。

2. **消退（Extinction）**：当 CS 不再伴随 UCS 出现时，条件反应逐渐减弱。消退不等于遗忘，原始学习仍然存在。

3. **自发恢复（Spontaneous Recovery）**：消退后经过一段时间，条件反应可能短暂地重新出现。

4. **泛化（Generalization）**：对与 CS 相似的刺激也产生条件反应。例如，被一只狗咬过后害怕所有的狗。

5. **分化（Discrimination）**：学会只对特定的 CS 做出反应，对相似但不同的刺激不反应。

### 在心理健康中的应用

**恐惧和焦虑的形成**：
- 一次车祸（UCS）引发恐惧（UCR），此后看到汽车（CS）也会感到恐惧（CR）
- 在学校被嘲笑（UCS）引发羞耻（UCR），此后对学校环境（CS）也会感到焦虑（CR）

**系统脱敏疗法**：
基于经典条件反射的消退原理，通过渐进地将个体暴露于恐惧刺激中（同时保持放松状态），来消退恐惧反应。这是治疗恐惧症的经典方法。

**情绪的条件化**：
- 特定音乐与过去的愉快经历关联，听到时会感到温暖
- 某种气味与童年记忆关联，闻到时会引发特定情绪

## 操作性条件反射

### 斯金纳的理论

操作性条件反射（Operant Conditioning）由 B.F. Skinner 系统研究和发展。与经典条件反射不同，操作性条件反射关注的是行为的后果如何影响行为再次发生的概率。

### 核心原理

行为之后出现的结果决定了该行为未来发生的频率：
- 如果结果是有利的（强化），行为频率增加
- 如果结果是不利的（惩罚），行为频率减少

### 强化与惩罚的四象限

| | 增加刺激 | 移除刺激 |
|---|---------|---------|
| **增加行为** | 正强化（给予奖励） | 负强化（移除不愉快刺激） |
| **减少行为** | 正惩罚（给予不愉快刺激） | 负惩罚（移除愉快刺激） |

### 强化程序表

1. **连续强化（CRF）**：每次行为出现都给予强化。适合学习新行为，但消退速度快。
2. **固定比率（FR）**：每做出固定次数的行为给予一次强化。如每完成 5 道题给一次表扬。
3. **变动比率（VR）**：不固定次数的行为后给予强化。如赌博机制。产生最高的行为频率和最强的抗消退能力。
4. **固定间隔（FI）**：在固定时间间隔后第一次出现的行为给予强化。
5. **变动间隔（VI）**：在不固定的时间间隔后给予强化。如不定期的检查奖励。

### 在心理健康中的应用

**回避行为的维持**：
- 焦虑→回避→焦虑暂时减轻（负强化）→回避行为被强化
- 这是焦虑障碍中回避行为持续存在的核心机制

**抑郁的行为模型**：
- 抑郁时积极行为减少→正面强化减少→更少积极行为（恶性循环）
- 行为激活疗法正是基于操作性条件反射原理设计的

## 社会学习理论

### 班杜拉的贡献

Albert Bandura 的社会学习理论（后发展为社会认知理论）扩展了行为主义，强调观察学习和认知过程在行为习得中的作用。

### 观察学习（替代学习）

个体不需要亲身经历强化，仅通过观察他人（榜样）的行为及其后果就能学习新行为。

**观察学习的四个过程**：
1. **注意过程（Attention）**：个体必须注意到榜样的行为。榜样的吸引力、能力、与观察者的相似性影响注意。
2. **保持过程（Retention）**：将观察到的行为以符号形式（言语、图像）存储在记忆中。
3. **动作再现过程（Reproduction）**：将记忆中的符号表征转化为实际行为。需要身体能力和练习。
4. **动机过程（Motivation）**：是否有动力去执行观察到的行为。取决于预期的强化或惩罚。

### 自我效能感（Self-Efficacy）

自我效能感是 Bandura 理论中的核心概念，指个体对自己成功执行特定行为能力的信念。

**四种效能信息来源（按影响力排序）**：
1. **直接经验（掌握经验）**：过去在类似任务上的成功经验是最强有力的效能来源
2. **替代经验**：观察与自己相似的人成功完成任务
3. **言语说服**：他人的鼓励和肯定
4. **生理和情绪状态**：积极的情绪和较低的焦虑水平提升效能感

**在青少年心理辅导中的应用**：
- 设计"小成功"体验来逐步提升自我效能感
- 分享同龄人成功克服困难的故事（替代经验）
- 真诚的鼓励和具体的肯定（"你上次处理得很好"比"你很棒"更有效）
- 教授放松技术来管理焦虑，间接提升效能感

### 交互决定论

Bandura 提出行为、个人因素（认知、情绪）和环境三者相互影响、互为因果：
- 个人因素影响行为选择和环境感知
- 行为改变环境，也反过来改变个人因素
- 环境提供行为的机会和约束，也影响个人因素

## 行为塑造和链接

### 行为塑造（Shaping）

行为塑造是通过强化连续近似行为来逐步建立目标行为的技术。

**实施步骤**：
1. 确定最终目标行为
2. 评估当前行为的起点水平
3. 设计从起点到目标的连续近似行为序列
4. 对每一个近似行为给予强化
5. 当前一个近似行为稳定后，提高标准，强化下一个更接近目标的行为

**示例**：帮助社交焦虑的青少年学会在班级发言
- 第一步：在小组内小声说话（强化）
- 第二步：在小组内正常音量说话（强化）
- 第三步：向同桌提问（强化）
- 第四步：在小组讨论中发言（强化）
- 第五步：在全班面前发言（强化）

### 行为链接（Chaining）

行为链接是将复杂行为分解为一系列简单步骤，按顺序逐步教授的技术。

**正链接**：从第一步开始，依次教授每个步骤
**逆链接**：从最后一步开始，逆向教授（适用于个体已掌握大部分步骤的情况）
**总任务呈现**：每次练习完整的任务链，在困难步骤提供额外帮助

## 正强化和负强化在情绪管理中的应用

### 情绪管理的行为策略

1. **正强化在情绪管理中的应用**
   - 当使用健康的情绪调节策略时给予自我奖励
   - 建立"情绪管理积分系统"：每次成功使用替代行为积累积分
   - 社会性强化：分享成功经验获得他人认可

2. **负强化在情绪管理中的应用**
   - 识别哪些行为是通过负强化维持的（如通过暴食暂时消除焦虑）
   - 用健康的应对行为替代不良行为，同样提供焦虑缓解的负强化效果
   - 渐进式暴露中，焦虑的自然消退本身就是强有力的负强化

3. **消退的应用**
   - 识别维持不良情绪行为的社会强化（如通过发脾气获得关注）
   - 有计划地消退不良行为，同时强化替代行为
   - 注意消退爆发（extinction burst）：行为在消退前可能暂时增强

### 应用原则

- **即时性**：强化应紧跟目标行为，延迟的强化效果减弱
- **一致性**：初期应持续强化，建立稳定的行为模式
- **个体化**：选择对个体有实际强化价值的强化物
- **自然化**：逐渐过渡到自然环境中的强化，确保行为的泛化和维持

## 习惯养成的行为学基础

### 习惯形成的行为模型

习惯是在特定情境线索下自动发生的行为模式，其形成遵循"线索-行为-奖励"的循环。

**习惯回路（Habit Loop）**：
1. **线索（Cue）**：触发行为的环境信号（时间、地点、情绪状态、 preceding 行为、他人）
2. **行为（Routine）**：被触发的习惯行为
3. **奖励（Reward）**：行为带来的满足感，强化整个回路

### 习惯改变的四个法则（James Clear）

**建立好习惯**：
1. 让它显而易见：设置明确的线索提醒
2. 让它有吸引力：将习惯与愉快体验配对
3. 让它容易：降低行为门槛，从两分钟版本开始
4. 让它令人满足：提供即时奖励和追踪

**戒除坏习惯**：
1. 让它不可见：移除环境中的触发线索
2. 让它没吸引力：关注坏习惯的负面后果
3. 让它困难：增加行为的摩擦力
4. 让它令人痛苦：设置即时的负面后果

### 在青少年情绪管理中的应用

**建立健康情绪习惯**：
- 将深呼吸练习与每天的固定时间（如起床后、睡前）配对
- 使用手机提醒作为正念练习的线索
- 情绪日记作为睡前习惯的一部分
- 运动与放学回家的路线配对（经过操场就去跑步）

**改变不良情绪习惯**：
- 识别焦虑时的自动化行为（如刷手机、咬指甲）
- 用替代行为填补（如焦虑时捏压力球代替刷手机）
- 调整环境减少触发（如睡前一小时将手机放在客厅）
- 使用习惯逆转训练（Habit Reversal Training）的原理
