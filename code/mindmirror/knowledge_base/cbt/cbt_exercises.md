# CBT 练习指南

## 概述

本指南提供了一系列结构化的 CBT 练习，帮助个体在实践中学习和应用认知行为疗法的核心技术。每个练习都包含详细的操作步骤、使用场景和注意事项。建议在治疗师指导下进行练习，也可以通过自助方式逐步掌握这些技能。

## 思维记录表练习

思维记录表（Thought Record）是 CBT 中最基础也最重要的工具之一，帮助个体系统地识别和分析自己的自动思维。

### 基础版：三栏思维记录

适用于初学者，帮助建立想法-情绪-行为的觉察能力。

| 栏目 | 说明 | 示例 |
|------|------|------|
| 情境 | 描述引发情绪的事件 | 老师在全班面前批评了我的作业 |
| 自动思维 | 脑海中闪过的想法 | "我太笨了，老师肯定觉得我不用心" |
| 情绪 | 感受到的情绪及强度 | 羞耻(80%)、焦虑(70%)、难过(60%) |

### 进阶版：五栏思维记录

在三栏基础上增加证据分析和替代思维。

| 栏目 | 说明 |
|------|------|
| 情境 | 客观描述触发事件 |
| 自动思维与情绪 | 记录想法和情绪强度 |
| 支持证据 | 列出支持自动思维的客观事实 |
| 反对证据 | 列出不支持自动思维的客观事实 |
| 替代思维与重新评估 | 生成平衡想法，重新评估情绪强度 |

### 完整版：七栏思维记录

在五栏基础上增加认知扭曲识别和行为计划。

**操作步骤**：
1. 每天选择一个引发强烈情绪的情境（情绪强度≥60%）
2. 尽可能详细地填写每一栏
3. 特别注意区分"想法"和"情绪"（"我觉得他在生气"是想法，"我感到害怕"是情绪）
4. 证据栏中只记录可验证的客观事实，不包含推测
5. 替代思维应是可信的、平衡的，而非盲目积极
6. 坚持练习至少两周，每天至少完成一条记录

### 练习要点

- **时机**：最好在情绪发生后尽快记录，以保证自动思维的准确性
- **频率**：初期每天 1-2 条，熟练后可根据需要灵活使用
- **常见困难**：初学者常混淆想法和情绪，需要反复练习区分

## 认知重构练习

认知重构是在思维记录基础上更深入地挑战和改变不合理思维的技术。

### 练习一：苏格拉底式提问

对自己提出以下引导性问题来挑战自动思维：

1. **证据检验**
   - "支持这个想法的证据是什么？这些证据可靠吗？"
   - "反对这个想法的证据是什么？"
   - "如果我的朋友有同样的想法，我会怎么对他说？"

2. **替代解释**
   - "还有没有其他可能的解释？"
   - "如果从一个旁观者的角度看，会怎么理解这件事？"
   - "一年后回头看这件事，我还会这样想吗？"

3. **后果评估**
   - "最坏的情况是什么？如果发生了，我能应对吗？"
   - "最好的情况是什么？"
   - "最现实的情况是什么？"

4. **行动导向**
   - "我现在可以做些什么来改善情况？"
   - "即使这个想法是真的，我可以怎么应对？"

### 练习二：思维法庭

将自动思维想象成一场法庭审判：
1. 你是检察官，列出支持这个想法的所有证据
2. 你是辩护律师，列出反对这个想法的所有证据
3. 你是法官，基于双方证据做出公正裁决
4. 写出"判决书"——一个更加平衡和客观的结论

### 练习三：去灾难化

当出现灾难化思维时，使用以下步骤：
1. 明确描述最坏的结果是什么
2. 评估这个最坏结果发生的概率（0-100%）
3. 即使最坏的结果发生了，想象你会如何应对
4. 回顾过去类似担忧的结果——它们真的像你想象的那么糟糕吗？
5. 形成更现实的预期

## 放松训练：渐进式肌肉放松

渐进式肌肉放松（Progressive Muscle Relaxation, PMR）由 Edmund Jacobson 开发，通过系统地紧张和放松不同肌群来减轻身体焦虑。

### 基本原理

焦虑和压力会导致肌肉紧张，而肌肉放松可以反过来降低焦虑水平。通过交替进行紧张和放松，个体可以学会识别和释放身体中的紧张感。

### 完整练习步骤

**准备**：找一个安静舒适的地方，坐下或躺下，闭上眼睛，做几次深呼吸。

**练习顺序（约 20 分钟）**：

1. **手部和前臂**：握紧拳头，保持 5 秒 → 缓慢松开，感受放松 15 秒
2. **上臂（肱二头肌）**：弯曲手臂，绷紧上臂 → 放下，放松
3. **上臂（肱三头肌）**：伸直手臂，绷紧上臂后方 → 放松
4. **前额**：抬高眉毛，皱紧前额 → 放松
5. **眼睛和面颊**：紧闭双眼，皱紧面颊 → 放松
6. **下颌和颈部**：咬紧牙关，将下巴向胸部方向压 → 放松
7. **肩部**：耸肩至耳朵位置 → 放下，放松
8. **肩胛骨**：将两肩胛骨向中间挤压 → 放松
9. **胸部**：深吸一口气，绷紧胸部 → 缓慢呼气，放松
10. **腹部**：收紧腹肌 → 放松
11. **臀部**：收紧臀部肌肉 → 放松
12. **大腿**：绷紧大腿肌肉 → 放松
13. **小腿**：脚尖向上翘，绷紧小腿 → 放松
14. **脚部**：蜷曲脚趾 → 放松

### 注意事项

- 每次紧张保持 5 秒，放松保持 15-20 秒
- 重点关注紧张和放松之间的对比感
- 如果某个部位有伤痛，跳过该区域
- 规律练习（每天 1-2 次）比偶尔长时间练习更有效
- 可以在焦虑初期使用简短版（只练习 4-5 个主要肌群）

## 正念呼吸练习

正念呼吸是正念减压（MBSR）的核心练习，帮助培养对当下体验的非评判性觉察。

### 基础正念呼吸（5 分钟版）

1. **准备姿势**：坐直但不要僵硬，双手放在膝盖上，双脚平放在地面
2. **注意力锚点**：将注意力集中在呼吸上，感受空气进入鼻腔的凉感和呼出的温热
3. **觉察走神**：当注意力飘走时（这一定会发生），温柔地注意到这一点
4. **非评判回归**：不加批评地将注意力带回呼吸，走神并回来本身就是练习
5. **结束**：慢慢将注意力扩展到整个身体，然后睁开眼睛

### 进阶练习：扩展觉察

在稳定呼吸觉察的基础上，逐步扩展觉察范围：
1. 呼吸感觉 → 身体整体感觉 → 声音 → 想法和情绪 → 开放觉察

### 三分钟呼吸空间（3-Minute Breathing Space）

适用于日常生活中的快速正念练习：
- **第一分钟（觉察）**：问自己"我现在的体验是什么？"注意想法、情绪和身体感觉
- **第二分钟（聚焦）**：将注意力集中到呼吸上，用呼吸作为锚点
- **第三分钟（扩展）**：将觉察从呼吸扩展到整个身体，感受身体作为一个整体的存在

### 练习建议

- 初学者从每天 5 分钟开始，逐渐增加到 15-20 分钟
- 固定时间和地点有助于建立习惯
- 走神不是失败，觉察走神并回来才是正念的核心
- 可以使用引导音频辅助初期练习

## 行为实验设计

行为实验（Behavioral Experiment）是 CBT 中最有力的技术之一，通过实际行动来检验和挑战不合理信念。

### 设计框架

1. **识别目标信念**
   - 明确要检验的信念，用 0-100% 评估对信念的相信程度
   - **示例**："如果我主动跟同学说话，他们一定会觉得我很烦。"（相信程度 85%）

2. **设计实验**
   - 制定具体的行动计划来检验这个信念
   - 明确"如果信念是对的，会发生什么"和"如果信念是错的，会发生什么"
   - **示例**：明天课间主动向一位同学打招呼并简短聊天

3. **预测结果**
   - 写下你预测会发生的事情
   - 设定判断标准：什么结果支持信念，什么结果不支持

4. **执行实验**
   - 按计划执行，尽可能收集实际发生的信息
   - 记录对方的实际反应和自己的感受

5. **反思结果**
   - 比较预测和实际结果
   - 重新评估对原始信念的相信程度
   - 形成新的、更平衡的信念

### 行为实验的类型

- **假设检验型**：直接测试某个预测是否成立
- **发现型**：探索"如果我这样做会发生什么"，没有特定预期
- **调查型**：通过询问他人来收集信息（如"你会怎么看待主动打招呼的人？"）

### 设计原则

- 从风险较低的实验开始
- 即使结果不完全如预期，也能从中学习
- 重复进行以巩固新的学习
- 记录每次实验的结果，建立"证据档案"

### 青少年适用的行为实验示例

| 信念 | 实验设计 |
|------|----------|
| "没人愿意和我做朋友" | 在午餐时主动坐到一群同学旁边 |
| "如果我提问，老师会觉得我蠢" | 在下节课上提一个问题 |
| "我不擅长任何运动" | 参加一次不熟悉的运动活动 |
| "如果我拒绝别人，他们就不理我了" | 礼貌地拒绝一个不想做的事情 |
