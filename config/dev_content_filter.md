# 开发者技术雷达 - 内容筛选标准

## ✅ 高价值内容（抓取 + 详细摘要）

### 1. 实战技术
- 新框架/工具发布（含使用示例）
- 性能优化技巧
- 最佳实践/设计模式
- 代码重构案例

### 2. 效率工具
- CLI 工具
- IDE 插件
- 自动化脚本
- 开发工作流优化

### 3. 技术深度
- 源码分析
- 架构设计
- 技术选型对比
- 踩坑总结

### 4. 趋势技术
- 新语言特性
- 新兴框架
- 有潜力的开源项目

## ❌ 低价值内容（仅标题或忽略）

- 高管变动/融资新闻
- 大会预告/回顾
- 产品发布公告（无技术细节）
- 市场分析报告
- 纯商业合作

## 摘要格式（高价值文章）

```
【技术类别】标题

核心价值：一句话说明能解决什么问题

关键要点：
• 要点 1（含具体数据/代码/工具名）
• 要点 2
• 要点 3

适用场景：什么情况下用得上

行动建议：是否需要深入了解/立即尝试
```
