## 设计说明

### 功能概述
本界面用于显示监控设备的录像诊断结果，包括：
1. **设备信息** - 显示设备名称和诊断状态
2. **存储统计** - 存储天数、录像总数、丢失数量、完整率
3. **丢失详情** - 时间轴展示 + 详细列表

### 视觉风格
- **风格**: 数据监控面板风格，清晰直观
- **主色**: #1890FF（科技蓝）
- **状态色**:
  - 正常：#52C41A（绿色）
  - 异常：#F5222D（红色）
  - 警告：#FAAD14（橙色）

### 交互效果
- ✅ 状态标签带呼吸动画
- ✅ 卡片悬停上浮效果
- ✅ 列表项悬停背景变化
- ✅ 按钮悬停阴影效果
- ✅ 箭头图标悬停平移

### 响应式适配
- 📱 手机 (<768px): 单列布局，统计卡片垂直排列
- 📱 平板 (768-1024px): 统计卡片 2 列布局
- 💻 桌面 (>1024px): 统计卡片 4 列布局
