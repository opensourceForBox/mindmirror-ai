## 1. 界面概述

**功能**: 用户进行 CBT（认知行为疗法）练习  
**入口**: 首页"CBT 练习" / 底部导航"练习"  
**目标**: 提供清晰、引导式的练习流程，帮助用户完成心理干预

## 4. 交互说明

### 4.1 练习流程

```
选择练习 → 查看介绍 → 开始练习 → 
  逐步完成（1-5 步） → 
  提交答案 → AI 生成反馈 → 
  显示完成界面 → 记录保存
```

### 4.2 步骤导航

```kotlin
// 步骤导航状态管理
@Composable
fun ExerciseSteps(
    currentStep: Int,
    totalSteps: Int,
    onStepChange: (Int) -> Unit
) {
    Column {
        // 进度条
        LinearProgressIndicator(
            progress = currentStep.toFloat() / totalSteps,
            modifier = Modifier.fillMaxWidth()
        )
        
        // 步骤指示
        Text(
            text = "步骤 $currentStep/$totalSteps",
            style = MindMirrorTypography.caption
        )
        
        Spacer(modifier = Modifier.height(16.dp))
        
        // 导航按钮
        Row(
            modifier = Modifier.fillMaxWidth(),
            horizontalArrangement = Arrangement.SpaceBetween
        ) {
            TextButton(
                onClick = { onStepChange(currentStep - 1) },
                enabled = currentStep > 1
            ) {
                Text("上一步")
            }
            
            Button(
                onClick = { onStepChange(currentStep + 1) },
                enabled = currentStep < totalSteps
            ) {
                Text("下一步")
            }
        }
    }
}
```

## 5. 练习类型详情

### 5.1 思维记录表（Thought Record）

**步骤**:
1. 描述情境（触发事件）
2. 记录自动思维
3. 识别情绪（类型 + 强度）
4. 寻找证据（支持/反对）
5. 生成替代思维
6. 重新评估情绪

**示例输入**:
```
情境：老板在会议上批评了我的方案

自动思维：
- "我能力不够"
- "我可能会失业"
- "大家都觉得我很差"

情绪：
- 焦虑 (80%)
- 悲伤 (60%)

支持证据：
- 老板确实提出了批评
- 方案有改进空间

反对证据：
- 老板也肯定了部分优点
- 过去也有成功的案例
- 批评不等于否定整个人

替代思维：
"这是一个改进的机会，不代表我能力不足"

重新评估：
- 焦虑 (40%) ↓
- 悲伤 (30%) ↓
```

### 5.2 行为激活（Behavioral Activation）

**步骤**:
1. 列出日常活动
2. 评估每项活动的愉悦感 (P) 和成就感 (M)
3. 识别回避的行为
4. 制定活动计划
5. 执行并记录

### 5.3 正念冥想（Mindfulness）

**步骤**:
1. 选择时长（5/10/20 分钟）
2. 调整姿势
3. 引导式呼吸练习
4. 身体扫描
5. 结束冥想

## 6. ViewModel

```kotlin
@HiltViewModel
class ExerciseViewModel @Inject constructor(
    private val exerciseRepository: ExerciseRepository,
    private val feedbackGenerator: FeedbackGenerator
) : ViewModel() {
    
    val uiState: StateFlow<ExerciseUiState> = _uiState.asStateFlow()
    private val _uiState = MutableStateFlow(ExerciseUiState())
    
    fun loadExercise(exerciseId: String) {
        viewModelScope.launch {
            val exercise = exerciseRepository.getExercise(exerciseId)
            _uiState.update {
                it.copy(
                    exercise = exercise,
                    currentStep = 1,
                    answers = mutableMapOf()
                )
            }
        }
    }
    
    fun updateAnswer(stepId: String, answer: String) {
        _uiState.update {
            it.copy(
                answers = it.answers.toMutableMap().apply {
                    this[stepId] = answer
                }
            )
        }
    }
    
    fun nextStep() {
        val state = uiState.value
        if (state.currentStep < state.exercise?.steps?.size!!) {
            _uiState.update { it.copy(currentStep = it.currentStep + 1) }
        } else {
            completeExercise()
        }
    }
    
    fun previousStep() {
        val state = uiState.value
        if (state.currentStep > 1) {
            _uiState.update { it.copy(currentStep = it.currentStep - 1) }
        }
    }
    
    private fun completeExercise() {
        viewModelScope.launch {
            _uiState.update { it.copy(isCompleting = true) }
            
            val state = uiState.value
            val result = exerciseRepository.completeExercise(
                exerciseId = state.exercise!!.id,
                answers = state.answers,
                moodBefore = state.moodBefore,
                moodAfter = state.moodAfter
            )
            
            // 生成 AI 反馈
            val feedback = feedbackGenerator.generate(
                exercise = state.exercise,
                answers = state.answers
            )
            
            _uiState.update {
                it.copy(
                    isCompleting = false,
                    isCompleted = true,
                    feedback = feedback,
                    result = result
                )
            }
        }
    }
}

data class ExerciseUiState(
    val isLoading: Boolean = false,
    val isCompleting: Boolean = false,
    val isCompleted: Boolean = false,
    val exercise: Exercise? = null,
    val currentStep: Int = 1,
    val answers: Map<String, String> = emptyMap(),
    val moodBefore: Emotion? = null,
    val moodAfter: Emotion? = null,
    val feedback: Feedback? = null,
    val result: ExerciseResult? = null,
    val error: String? = null
)
```
