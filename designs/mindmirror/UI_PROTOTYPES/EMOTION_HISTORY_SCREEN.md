## 3. 核心组件

### 3.1 情绪趋势图

使用折线图展示情绪变化趋势，支持缩放和拖动。

**规格**:
- 高度：200dp
- X 轴：日期/时间
- Y 轴：效价 (-1 到 1)
- 颜色：根据情绪类型变化

### 3.2 情绪分布条

```
😊 开心     ▓▓▓▓▓▓▓░░░  35%
😌 平静     ▓▓▓▓▓▓▓▓░░  40%
😰 焦虑     ▓▓▓░░░░░░░  15%
😢 悲伤     ▓▓░░░░░░░░  10%
```

### 3.3 情绪日历

热力图形式展示每日主导情绪，类似 GitHub 贡献图。

**颜色图例**:
- 😊 积极：绿色系
- 😌 平静：蓝色系
- 😐 中性：灰色系
- 😰 焦虑：紫色系
- 😢 悲伤：深蓝灰色系
- 😠 愤怒：红色系

## 4. 交互说明

### 4.1 图表交互

- **点击数据点**: 查看该时间点的情绪详情
- **双指缩放**: 缩放图表时间范围
- **左右滑动**: 切换时间范围

### 4.2 列表交互

- **下拉刷新**: 更新最新数据
- **点击记录**: 查看情绪详情
- **左滑记录**: 删除选项

### 4.3 筛选功能

```
筛选条件：
- 情绪类型：多选
- 时间范围：日/周/月/自定义
- 强度：全部/高/中/低
- 来源：手动记录/AI 识别
```

## 5. ViewModel

```kotlin
@HiltViewModel
class EmotionHistoryViewModel @Inject constructor(
    private val emotionRepository: EmotionRepository
) : ViewModel() {
    
    val uiState: StateFlow<EmotionHistoryUiState> = _uiState.asStateFlow()
    private val _uiState = MutableStateFlow(EmotionHistoryUiState())
    
    init {
        loadEmotionHistory(TimeRange.LAST_7_DAYS)
    }
    
    fun loadEmotionHistory(timeRange: TimeRange) {
        viewModelScope.launch {
            _uiState.update { it.copy(isLoading = true) }
            
            try {
                val records = emotionRepository.getRecords(
                    startDate = timeRange.startDate,
                    endDate = timeRange.endDate
                )
                
                val trend = calculateTrend(records)
                val distribution = calculateDistribution(records)
                val insights = generateInsights(records)
                
                _uiState.update {
                    it.copy(
                        isLoading = false,
                        records = records,
                        trend = trend,
                        distribution = distribution,
                        insights = insights
                    )
                }
            } catch (e: Exception) {
                _uiState.update { 
                    it.copy(
                        isLoading = false,
                        error = e.message
                    ) 
                }
            }
        }
    }
    
    fun deleteRecord(recordId: String) {
        viewModelScope.launch {
            try {
                emotionRepository.deleteRecord(recordId)
                _uiState.update {
                    it.copy(
                        records = it.records.filter { r -> r.id != recordId }
                    )
                }
            } catch (e: Exception) {
                _uiState.update { it.copy(error = e.message) }
            }
        }
    }
}

data class EmotionHistoryUiState(
    val isLoading: Boolean = false,
    val records: List<EmotionRecord> = emptyList(),
    val trend: EmotionTrend? = null,
    val distribution: EmotionDistribution? = null,
    val insights: List<Insight> = emptyList(),
    val error: String? = null
)
```

## 6. 数据导出

### 6.1 导出格式

支持导出为 PDF 或 CSV 格式。

**PDF 报告包含**:
- 情绪趋势图
- 情绪分布图
- 文字洞察
- 详细记录列表

### 6.2 导出流程

```
点击"导出报告" → 
  选择时间范围 → 
  选择格式 (PDF/CSV) → 
  生成报告 → 
  分享/保存
```
