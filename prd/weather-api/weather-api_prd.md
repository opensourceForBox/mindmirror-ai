# 产品需求文档 - 天气查询 API

## 背景与目标
创建一个天气查询 API 服务，支持按城市查询当前天气和预报。

## 用户故事
1. 作为用户，我想要查询当前天气，以便了解实时天气状况
2. 作为用户，我想要查询未来 3 天预报，以便规划出行
3. 作为用户，我想要查询多个城市，以便比较不同地区天气

## 验收标准
- Given 用户发送城市名，When 城市存在，Then 返回当前天气数据
- Given 用户请求预报，When 城市存在，Then 返回 3 天预报数据
- Given 用户查询不存在城市，Then 返回错误提示

## 技术栈
- 后端：Python Flask
- 数据源：模拟数据（后续可接入真实 API）
- API 风格：RESTful JSON
