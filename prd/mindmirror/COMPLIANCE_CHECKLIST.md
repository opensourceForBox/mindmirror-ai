## 2. 数据分类与分级

### 2.1 数据类型识别

| 数据类型 | 示例 | 敏感级别 | 处理要求 |
|----------|------|----------|----------|
| 个人基本信息 | 姓名、手机号、邮箱 | 中 | 加密存储，最小化收集 |
| 健康信息 | 情绪状态、心理评估 | 高 | 加密 + 脱敏，单独同意 |
| 生物识别信息 | 面部特征、语音特征 | 极高 | 本地处理，不存储原始数据 |
| 对话记录 | 心理咨询对话 | 高 | 端到端加密，可删除 |
| 危机信号 | 自伤/自杀倾向 | 极高 | 即时处理，最小化留存 |

### 2.2 数据生命周期管理

#### 收集阶段
- [ ] 明确告知收集目的、方式、范围
- [ ] 获取用户单独同意（敏感信息）
- [ ] 提供拒绝选项（不影响基本功能）
- [ ] 未成年人需监护人同意

#### 存储阶段
- [ ] 分类分级存储
- [ ] 加密存储（AES-256）
- [ ] 访问控制（最小权限原则）
- [ ] 存储期限明确（对话记录≤2 年）

#### 使用阶段
- [ ] 目的限制（仅用于约定用途）
- [ ] 去标识化处理
- [ ] 访问日志记录
- [ ] 禁止未经授权的共享

#### 删除阶段
- [ ] 用户可申请删除全部数据
- [ ] 账户注销后 30 天内删除
- [ ] 备份数据同步删除
- [ ] 删除记录可审计

## 3. 技术安全措施

### 3.1 传输安全
- [ ] TLS 1.3 加密传输
- [ ] HTTPS 强制启用
- [ ] 证书固定（移动端）
- [ ] API 签名验证

### 3.2 存储安全
- [ ] 数据库加密（TDE）
- [ ] 敏感字段应用层加密
- [ ] 密钥管理系统（KMS）
- [ ] 定期密钥轮换

### 3.3 访问控制
- [ ] 身份认证（JWT + 双因素）
- [ ] 权限管理（RBAC）
- [ ] 会话管理（超时自动登出）
- [ ] 异常访问检测

### 3.4 审计与监控
- [ ] 操作日志完整记录
- [ ] 日志保留≥6 个月
- [ ] 异常行为告警
- [ ] 定期安全审计

### 3.5 生物特征保护
- [ ] 面部特征本地提取（不上云）
- [ ] 原始视频不存储
- [ ] 特征向量加密存储
- [ ] 用户可随时删除生物特征

## 4. 用户权利保障

### 4.1 知情权
- [ ] 隐私政策清晰易懂
- [ ] 数据收集使用告知
- [ ] 第三方共享告知
- [ ] 政策变更通知

### 4.2 访问权
- [ ] 用户可查看个人数据
- [ ] 数据导出功能（JSON/CSV）
- [ ] 处理记录可查询

### 4.3 更正权
- [ ] 用户可修改个人信息
- [ ] 错误数据更正流程

### 4.4 删除权（被遗忘权）
- [ ] 一键删除账户及数据
- [ ] 删除确认机制
- [ ] 删除后不可恢复告知

### 4.5 撤回同意权
- [ ] 可随时撤回授权
- [ ] 撤回后停止处理
- [ ] 不影响撤回前处理效力

### 4.6 限制处理权
- [ ] 可申请限制数据处理
- [ ] 限制期间仅存储不处理
